# Graph-of-Shots — Documentation Index

This directory contains reference documentation for the repository. Start
with whichever matches your task:

| File | Purpose |
|---|---|
| [`repo_structure.md`](repo_structure.md) | What lives where — code, scripts, logs, presentation assets |
| [`results.md`](results.md) | Observed numbers (v1, v2, ablations) with paths to the raw logs that produced them |
| [`reproduce.md`](reproduce.md) | Step-by-step commands to reproduce every result from a fresh clone |
| [`changelog.md`](changelog.md) | Chronological list of commits, what each fixed, and their commit hashes |

The top-level [`../README.md`](../README.md) is the short onboarding
document (what the project is, how to install, how to run one example);
everything deeper lives here.
